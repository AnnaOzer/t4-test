когда в экшене пишем аргументы
фреймворк смотрит и

public function actionFoo($id)
в $id будет подставлено первое из
- параметр роута id
- $_POST['id']
- $_GET['id']
- значение по умолчанию

важная особенность
через type-hinting работает приведение типов!
включая  даже приведение к классу (лишь бы он реализовывал интерфейс
IArrayable)
должны быть методы фромэррэй и туэррэй
будет пытаться привести к нужному классу
из массива
где это применять
построить на страничке фильтры
пост или гет данные от пользователя- что он выбрал
получить и преобразовать в объект

public function actionFoo($id)
public function actionBar($id = 0)
public function actionBaz(int $id)
public function actionFoo(Std $data)

считается хорошим тоном писать значения по умолчанию для аргументов экшена
