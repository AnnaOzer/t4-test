Проброс методов
если есть свойство age
есть метод setAge
который устанавливает значение этому свойству

а если вызову у коллекции метод, о котором она не знает?
то этот метод будет вызван у ВСЕХ элементов этой коллекции.
Вызов незнакомого методы будет проброшен каждому элементу.
и метод save() будет вызван у каждой модели, находящейся в коллекции

жестоко если вызвать по ошибке
помогает если надо оптом обработать данные
например чтобы не вызывать save  в цикле
