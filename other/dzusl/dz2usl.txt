1. Создайте файл, возвращающий массив новостей. Каждая новость - это тоже массив, с элементами, допустим,
title, text, image. Это будет ваша "база данных"
2. В папке protected/Models разместите класс App\Models\Article extends Std, который будет представлять
из себя одну конкретную новость.
3. В папке protected/Models разместите класс App\Models\News extends Config, который будет представлять
собой "базу данных" из пункта 1. Продумайте нужные методы,
как минимум это public function findAll():array и public function findOne($id):Article
Возможно еще какие-то нужные вам.
4. Создайте контроллер Новости и экшны Все новости и Одна новость.
Придумайте красивые URL и правила роутинга для них.
Сделайте страницы "Все новость" и "Одна новость с указанным id"
5. Выведите последнюю новость на главной странице сайта через блок
6.* Сделайте экшн выводящий форму добавления новости и экшн,
непосредственно добавляющий новость с прототипом public function add(Article $data)
